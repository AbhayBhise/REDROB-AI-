# Archived Baseline Experiments

This directory contains exploratory notebooks, data cleaning scripts, baseline models, and earlier pipeline iterations from the initial days of the hackathon.

These files are retained solely for reference to show the EDA (Exploratory Data Analysis) and baseline modeling process:
* **`src/`**: Baseline feature extraction and pipeline code.
* **`notebooks/`**: Baseline data plotting notebook.
* **`data/`**: Intermediate CSVs and plots from the baseline runs.

**Note:** The production candidate ranking engine (`rank.py`), offline embedding generator (`precompute.py`), and the Streamlit demo application (`sandbox_app.py`) are located in the **repository root**. Please refer to the root `README.md` for execution instructions.
